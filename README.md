# Anti-Share Bypass

A standalone executable that:
1. Shows an "Inject" button on startup
2. Deletes all logs when clicked
3. Prevents re-injection until device restart

## Features
- Deletes all logs in %TEMP% and other locations
- Screen share bypass
- One-time injection only

## Usage
1. Run the executable
2. Click "Inject" to delete all logs
3. Restart device to inject again
